void * cla3p::i2malloc ( bulk_t size)

The default cla3p allocator.

Allocates size bytes of uninitialized storage.

Parameters
[in]sizeThe requested size in bytes.
Returns
On success, a pointer to the allocated space. Otherwise a null pointer.

The default cla3p allocator.

Allocates (nmemb x size) bytes of storage and initializes it to all bits zero.

Parameters
[in]nmembThe number of members to allocate.
[in]sizeThe size of each member in bytes.
Returns
On success, a pointer to the allocated space. Otherwise a null pointer.
