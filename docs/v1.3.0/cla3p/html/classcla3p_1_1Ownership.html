◆ owner()

bool cla3p::Ownership::owner ( ) const

Test whether object is the owner of its contents.

If the ownership flag is true, the object is responsible for deallocation of its contents.
The content memory is deallocated on object destruction.

Returns
whether the object is the owner of its contents.

◆ unbind()

void cla3p::Ownership::unbind ( )

Unties data from object.

Makes the object no longer responsible for content deallocation.
Use with caution as it may lead to memory leaks if the contents are not deallocated by the user.
Deallocate auto-allocated data using i_free().
